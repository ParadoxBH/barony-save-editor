INSTRUÇÕES DE USO – BARONY SAVE EDITOR

O programa Barony Save Editor foi desenvolvido por ParadoxBH. Este software NÃO é oficial e pode conter erros ou bugs. Ele serve apenas para ajudar na manipulação e edição de saves do jogo Barony.

O QUE EXISTE DENTRO DO ARQUIVO ZIP:
Dentro do ZIP existem arquivos de save separados para cada jogador que estava na sala.
Os arquivos estão nomeados no seguinte formato:

{nome do jogador}__{nome do arquivo de save}

Exemplo:
Pedro__savegame3_mp.baronysave

O QUE VOCÊ PRECISA FAZER:

Escolha o arquivo que pertence ao seu jogador.

Renomeie o arquivo removendo a parte "{nome do jogador}__" do nome.
O nome deve ficar apenas como o nome original do save.

Exemplo:
De → Pedro__savegame3_mp.baronysave
Para → savegame3_mp.baronysave

LOCAL ONDE O ARQUIVO DEVE SER COLOCADO:
Coloque o arquivo renomeado na seguinte pasta:

...\SteamLibrary\steamapps\common\Barony\savegames

FORMATOS DE NOME OBRIGATÓRIOS PARA O JOGO RECONHECER O SAVE:

Singleplayer:
savegame{numero}.baronysave
Exemplo: savegame1.baronysave

Multiplayer:
savegame{numero}_mp.baronysave
Exemplo: savegame2_mp.baronysave

IMPORTANTE SOBRE O JOGO ABERTO:
Não é necessário fechar e abrir o jogo toda vez que editar os arquivos.
O procedimento já funciona desde que você esteja na tela inicial do jogo, antes da seleção de save.
Apenas certifique-se de voltar para o menu principal antes de alterar qualquer arquivo na pasta de save.

Após isso, o save deve aparecer normalmente como se fosse criado pelo próprio jogo.
Se não funcionar, tente usar outro número no nome do arquivo ou verifique se ele está na pasta correta.
